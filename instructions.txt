CSS Selectors

1. Element Selector

			element{
						property:value;
			}

2. ID Selector   #
		Syntax
		element#value{
					property:value;
		}
		
		Example
		div#a1{
					color:red;
		}

3. Psudo Class Selector  :

		element:hover{
		}
